# Guardião Florestal Digital - IoT & Blockchain

Sistema inteligente de monitoramento e detecção de focos de incêndio para a conservação da biodiversidade amazônica.

## Sobre o Projeto
O **Guardião Florestal Digital** utiliza tecnologia IoT para criar uma rede de proteção na floresta. O sistema baseia-se em módulos equipados com sensores e tags NFC para rastreabilidade de árvores e monitoramento em tempo real.

Este repositório contém o **Simulador de Propagação de Alertas**, que demonstra como os nós da rede (Mesh) se comunicam ao detectar uma anomalia, propagando o sinal e registrando eventos.

## Funcionalidades
- **Simulação de Incêndio:** Clique em qualquer árvore para iniciar um alerta.
- **Rede Mesh:** Visualização da propagação do sinal entre os módulos vizinhos.
- **Registro em Blockchain:** Simulação de geração de hash para garantir a imutabilidade dos dados de alerta.
- **Interface Responsiva:** Painel de status lateral com logs em tempo real.

## Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript Moderno.
- **Conceitos de Hardware:** ESP32, Sensores de fumaça/chama, NFC.
- **Protocolos:** Simulação de rede Mesh e registros Blockchain.
